How to Find the Source URL for an Image

Learn how to get an image's source url.

Any image that you want to use for deviantART skins must first be uploaded on a website. If you want to use an image of your own, you can first upload your image onto an image hosting site such as dA's Sta.sh

The image URL is NOT the URL of the page you found your image on. To find the image URL, you must right click on an image and select the option that is most similar to "Copy Image URL".

The image URL is now on your clipboard, and you can select paste or press CTRL + V to use your image URL.
