Profile CSS Tutorial

Step-by-step tutorial for adding CSS to your profile through the use of a Featured Deviation widget. Please note that you must be a premium member to be able to use this feature.

First, go to your Sta.sh and click the large Compose in Sta.sh Writer button on the right side of the screen.

Click the Edit button in the top menu and select Switch to HTML Mode. You can also press CTRL + E.

Make sure to add an appropriate title to your jounal (this is what your widget will be called), and paste your HTML Code into the main text box of you journal.

Click the small, blue Edit Skin link on the right side of the screen to open up the CSS. Name your skin and paste your CSS Code into the main Skin CSS text box. Click the green Save button to save your skin.

Click the green Done button in the top right corner, and then press Submit.

On the submit page, select the Literature category instead of the "My Journal" category. Click the green Submit Now button to submit your directory as a literature submission.

Go to your profile page and add a Featured Deviation widget to your page. Click the pencil at the top right corner of the widget to edit it. Paste your Widget Code into the text box, and then click the Choose A Deviation button.

Select your profile directory in the pop up and click the green OK button. Save your Featured Deviation Widget.

Still confused? Feel free to contact SimplySilent for additional help.
